## Project

Tiny Rust CLI that generates a "NO AI" SVG badge (`badge.svg`) using the [`badgelib`](https://crates.io/crates/badgelib) crate. The badge is meant to be embedded in other projects' READMEs to signal the project was authored by a human, not generated by AI.

`badge.png` is a manually maintained PNG fallback for contexts where SVG is not supported — it is not produced by the binary.

## Common commands

```sh
cargo build                  # build debug binary
cargo run                    # build + run; rewrites ./badge.svg in repo root
./target/debug/no-ai-badge   # run already-built debug binary
cargo fmt                    # format
```

There are no tests and no CI configuration in this repo.

## Architecture

Single-file program (`src/main.rs`): constructs a `badgelib::Badge`, sets label/value/colors, calls `.to_svg()`, and writes the result to `./badge.svg` relative to the current working directory. Note that `cargo run` from the repo root will overwrite the committed `badge.svg`.

Any change to the badge's appearance (text, colors, version metadata) is made by editing the builder chain in `main.rs`.
